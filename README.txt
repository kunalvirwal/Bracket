Bracket is a Freelancing and Recruiting platform which allows new freelancers to explore job opportunities.
Recruiters can find new freelacers and offer jobs.

To run the project
>>> cd to the project repository
>>> Run the virtual env using  (.\env\Scripts\activate)
>>> Run the command (python manage.py makemigrations)
>>> Run the command (python manage.py migrate)
>>> Run the command (python manage.py runserver)
>>> Signup to make a new account and enjoy!
